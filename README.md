# Dimos Contribution Repository

Skills and scripts for [Dimos](https://github.com/dimensionalOS/dimos).

Dimos is a universal framework for AI-native generalist robotics.

## Contents

- Home Assistant skill, allows turning on/off lights
- NOSTR skill, allows chatting with the robot using a [NOSTR](https://nostr.com/) client
- Radar point cloud streaming for the xwr68xxisk](https://github.com/juhasch/xwr68xxisk)
- Local TTS using Chatterbox
- Local STT using FasterWhisper

This all is work in progress...
